# AI Career Guidance System

An AI-powered web application that provides **personalized career insights and growth tools** based on a user’s professional background.

## Overview

The AI Career Guidance System collects user details such as **industry, specialization, experience, skills, and professional bio** after login. Using this data, it generates a **customized dashboard** with AI-driven career insights and recommendations.

## Features

- **Authentication** using Clerk  
- **Personalized Career Dashboard**:
  - Market Overview
  - Industry Growth
  - Demand Level & Top Skills
  - Salary Ranges
  - Recommended Upskilling Areas
- **Interactive AI Chatbot** for career guidance
- **Growth Tools**:
  - Resume Builder
  - Interview Practice Quiz with history and graphical performance reports
  - AI Cover Letter Generator based on job details

## Tech Stack

- **Clerk** – Authentication & user management  
- **Next.js** – Full-stack React framework  
- **Shadcn** – Frontend Design
- **Neon PostgreSQL** – Serverless PostgreSQL database  
- **Inngest** – Background jobs and workflows
- **Gemini API** – AI-powered content generation
